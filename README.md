# 🏆 Eternity Sports Event – Anchoring Script

A beautifully designed **HTML-based anchoring script** for the **Eternity Sports Event 2025–2026 Prize Distribution Ceremony**.
This project provides a complete, ready-to-use script for event anchors along with an elegant UI layout for smooth presentation.

---

## 📌 Overview

This project is a **single-page HTML script** that combines:

* 🎤 Anchoring dialogues (Anchor 1 & Anchor 2)
* 🏅 Prize distribution structure
* 🎨 Stylish UI with animations and themes
* 🏆 Event-wise categorization (Cricket, Basketball, Football, etc.)
* 👦👧 Separate sections for Boys & Girls categories

It is designed to help anchors deliver a **professional, engaging, and energetic stage performance**.

---

## ✨ Features

* 🎭 **Dual Anchor Format**
  Clearly structured dialogues for Anchor 1 & Anchor 2

* 🎨 **Modern UI Design**

  * Gradient themes per sport
  * Smooth animations
  * Clean typography

* 🧩 **Modular Sections**

  * Opening Ceremony
  * Sports-wise Prize Distribution
  * Closing Ceremony (if extended)

* 📊 **Pre-filled & Editable Data**

  * Winners, Runner-ups
  * Placeholder fields (`[________]`) for easy updates

* 📱 **Responsive Design**
  Works on laptops, projectors, and mobile screens

## 🚀 How to Use

1. **Download or Clone the Project**

   ```bash
   git clone <your-repo-link>
   ```

2. **Open the File**

   * Double-click the HTML file
     OR
   * Open with any browser (Chrome, Edge, etc.)

3. **Edit Content (Optional)**

   * Replace placeholders like:

     ```
     [_____________ — to be filled]
     ```
   * Update team names, winners, or script lines

4. **Use During Event**

   * Display on laptop/projector
   * Anchors can read directly from screen

## 🎤 Script Flow

### 🔹 Opening

* Welcome speech

* Event kickoff

### 🔹 Sports Sections

* Introduction lines

* Prize announcements:

  * 🥉 2nd Runner Up
  * 🥈 Runner Up
  * 🥇 Winner

### 🔹 Closing (Extendable)

* Gratitude speech
* Event conclusion
